Első lépés
A program elindítása után forrás mappát kell választani, amelyben megtalálhatóak a képek
Második lépés
Ez után van lehetőségünk:
-Átméretezni a képet
-Elforgatni
-Kivágni egérrel, vagy pixelszám megadással
-A mentett kép minőségét állítani
-A beállított paramétereket egy, vagy az összes képre alkalmazni
-Formátumot módosítani
-Mentés helyét megadni
